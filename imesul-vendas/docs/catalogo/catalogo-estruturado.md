# Catálogo IMESUL - Estrutura para o Site

## Fonte analisada

- Arquivo: `catalogo-imesul-2024.pdf`
- Total de páginas: 15
- Formato: catálogo diagramado em imagens, sem camada de texto pesquisável
- Escopo deste documento: organizar categorias, subcategorias e exemplos comerciais; as tabelas completas de medidas, espessuras e pesos permanecem no PDF

## 1. Tubos metálicos

### Subcategorias

- Tubos retangulares
- Tubos quadrados
- Tubos redondos

### Exemplos de produtos

- Tubo retangular em diferentes combinações de largura, altura e espessura
- Tubo quadrado em diferentes seções e espessuras
- Tubo redondo em diferentes diâmetros e espessuras

### Aplicações sugeridas para o site

- Portões e grades
- Estruturas leves
- Serralheria
- Móveis e equipamentos metálicos
- Cercamentos e aplicações rurais

### Observações para implementação

- No site, o termo comercial `Metalon` pode funcionar como alias ou filtro para tubos quadrados e retangulares.
- O formulário futuro deve pedir formato, dimensões da seção, espessura e quantidade.
- As medidas devem ser cadastradas posteriormente como variações, sem transcrever automaticamente as tabelas do PDF.

### Aplicações sugeridas para o site

- O site deve tratar peso como dado técnico futuro, sem cálculo automático nesta fase.

## 4. Laminados

### Subcategorias

- Cantoneiras de abas iguais
- Barras chatas
- Barras quadradas
- Barras redondas

### Exemplos de produtos

- Cantoneira para reforço, apoio e montagem
- Barra chata para fabricação e serralheria
- Barra quadrada para estruturas e componentes
- Barra redonda para fabricação, grades e elementos mecânicos

### Aplicações sugeridas para o site

- Estruturas metálicas
- Serralheria
- Grades e portões
- Reforços e suportes
- Fabricação industrial

### Observações para implementação

- `Laminados` pode ser a categoria técnica, com `Cantoneiras` e `Barras` como entradas comerciais independentes.
- Para barras, o formulário deve permitir escolher formato: chata, quadrada ou redonda.
- Para cantoneiras, considerar aba e espessura como futuras variações.

- A disponibilidade real deve ser confirmada pela equipe comercial.

## 7. Perfis para serralheria

### Subcategorias por aplicação

- Perfis para portões de elevação
- Perfis para portas de aço
- Perfis para janelas de aço

### Exemplos mostrados

- Portões basculantes e de elevação
- Portas de enrolar e portas metálicas
- Portas sociais metálicas
- Janelas, grades e venezianas de aço

### Observações para implementação

- A página funciona mais como referência de aplicações do que como tabela de SKUs.
- Vincular esta categoria aos projetos `Portão` e `Serralheria`.
- Antes de criar produtos individuais, confirmar com vendas quais perfis e componentes são efetivamente comercializados separadamente.

## Estrutura de dados sugerida

Cada produto ou família poderá evoluir para uma estrutura como:

```js
{
  id: "telha-trapezoidal-40",
  categoryId: "telhas-metalicas",
  name: "Telha Trapezoidal 40",
  aliases: ["telha galvalume", "telha perfil 40"],
  description: "Telha metálica para coberturas comerciais, industriais e rurais.",
  applications: ["cobertura", "galpao", "rural"],
  optionFields: ["comprimento", "espessura", "quantidade"],
  availabilityNote: "Medidas e disponibilidade sujeitas à confirmação comercial."
}
```

## Pontos que exigem validação comercial

- Quais itens do PDF continuam ativos no catálogo atual.
- Quais medidas e espessuras ficam disponíveis em cada unidade.
- Quais produtos são vendidos por barra, peça, chapa, metro ou embalagem.
- Quais marcas podem ser apresentadas publicamente.
- Diferença comercial entre `Metalon` e os tubos quadrados/retangulares.
- Existência de bobinas ou bobininhas, pois não aparecem como uma categoria explícita neste PDF.
- Inclusão de produtos atuais que não constam no catálogo de 2024.
- Contatos e unidades que devem receber cada tipo de solicitação.
