# Glossario

- **Server Component:** componente renderizado no servidor, sem estado do navegador.
- **Client Component:** componente hidratado para controlar selecoes, eventos e formularios.
- **GSAP:** biblioteca de animacao usada no ecossistema IMESUL, principalmente no institucional.
- **ScrollTrigger:** plugin que sincroniza uma animacao com a rolagem.
- **Lenis:** recurso de rolagem suave usado pelo site institucional.
- **Framer Motion:** biblioteca de transicoes declarativas em React.
- **LazyMotion:** carregamento reduzido dos recursos do Framer Motion.
- **CSP:** politica que restringe origens de scripts, imagens e conexoes.
- **Turbopack:** empacotador usado pelo Next.js.
- **WebP:** formato das imagens otimizadas do catalogo.
- **WebM:** formato compacto de video usado no institucional.
- **Fallback:** valor ou recurso alternativo usado quando o principal nao existe.
- **Hydration:** conexao do React com o HTML entregue pelo servidor.
- **Viewport:** area visivel da pagina.
- **Lazy loading:** carregamento adiado de um recurso fora da area visivel.
