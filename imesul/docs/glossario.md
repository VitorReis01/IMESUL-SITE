# Glossario

- **Server Component:** componente renderizado no servidor, sem JavaScript proprio no navegador.
- **Client Component:** componente hidratado no navegador para usar estado, eventos ou APIs web.
- **GSAP:** biblioteca usada nas animacoes mais precisas da pagina.
- **ScrollTrigger:** plugin do GSAP que liga animacoes a posicao da rolagem.
- **Lenis:** biblioteca que suaviza a rolagem sem substituir o conteudo nativo.
- **Framer Motion:** biblioteca das entradas e transicoes declarativas em React.
- **LazyMotion:** forma de carregar apenas os recursos do Framer Motion usados pelo site.
- **CSP:** politica que limita as origens autorizadas a executar ou carregar recursos.
- **Turbopack:** empacotador usado pelo Next.js no desenvolvimento e no build.
- **WebP:** formato compacto usado nas imagens do site.
- **WebM:** formato principal do video institucional.
- **Fallback:** alternativa usada quando o recurso preferencial nao esta disponivel.
- **Hydration:** etapa em que o React conecta eventos ao HTML entregue pelo servidor.
- **Viewport:** area visivel da pagina no dispositivo.
- **Lazy loading:** carregamento adiado ate o recurso se aproximar da area visivel.
